# s_expression
Utilities for Handling of S-Expression Lisp/Scheme-Like forms and parsing of KIF, GDL, PDDL, CLIF

```prolog

:- pack_install('https://github.com/TeamSPoon/s_expression.git').


:- use_module(library(sexpr_reader)).

```
